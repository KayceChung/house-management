Hướng Dẫn & Lưu Ý
  • URL Deploy: Phải là URL từ Deployment chính thức, không phải URL preview
  • CORS: Google Apps Script tự động hỗ trợ CORS nếu bạn sử dụng redirect: follow
  • Lỗi 405: Xảy ra khi không có hàm doPost(e) - Đã sửa xong ✅
  • Version mới: Sau mỗi sửa code, phải tạo New Deployment
  • JSON từ POST: Gửi JSON trong body: JSON.stringify(payload) với header Content-Type: application/json
